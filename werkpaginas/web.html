Project Web

Student Budget - Hogeschool van Amsterdam - 2020

terug naar home

De opdracht

Project Web was een invidueel project waar we een multi-device applicatie moesten ontwerpen voor op reis. De eindoplevering is een prototype waar je duidelijk kan zien wat je voor, tijdens en na de reis met de applicatie kan en een design rationale.

Mijn eind concept is de app 'Student Budget' geworden. Met deze app kan je een budget instellen voor je reis en dit budget ook gemakkelijk bijhouden. Verder krijg je ook tips over leuke budget spots. Via de link kun je mijn prototype bekijken.

Naast een prototype moesten we ook een design rationale opleveren. In deze design rationale geef ik meer informatie over het gebruik van de app.
